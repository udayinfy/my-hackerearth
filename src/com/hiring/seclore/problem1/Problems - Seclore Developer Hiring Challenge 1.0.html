Array Transformation

Given two arrays A and B of N integers. Also, M transformations are given in the form of [xyw] representing that the element x can be transformed to the element y or vice versa with the cost w. Now, you have to transform the array A to the array B using these transformations with the minimal cost.

Note: Please go through the sample explanation carefully for better understanding of the problem.

Input Format

The first line of the input contains a single integer N, the total number of elements in the array.
The next line of the input contains N space-separated integers representing the elements of the array A.
The next line of the input contains N space-separated integers representing the elements of the array B.
The next line of the input contains a single integer M, the total number of transformations.
Then M lines follow, each containing three space-separated integers x, y and w representing that the element x can be transformed to the element y or vice versa with the cost w.

Output Format

In the single line of the output print the minimal cost if it is possible to transform the array A to the array B else if it is not possible print 1.

Constraints

1N,M5000
1Ai,BiN
1w109

Sample Input
7
1 2 4 3 4 7 5
5 2 3 2 7 2 6
6
1 6 20
1 5 10
2 4 5
4 3 50
4 7 12
3 7 18
Sample Output
134
Explanation

           

1.1 to 5=10
2.2 to 2=0
3.4 to 3=30
4.3 to 2=35
5.4 to 7=12
6.7 to 2=17
7.5 to 6=30

Total cost: 10+0+30+35+12+17+30=134

Maximum Co-prime Pair

Given two arrays A and B of N elements. Now, you have to find out two elements x and y such that:
1. Element x belongs to the array A and element y belongs to the array B
2. Elements x and y are co-prime i.e. gcd(x,y)=1
3. Sum of x and y is maximum possible

Input Format
The first line of the input contains a single integer N, the total number of elements in the array.
The next line contains N space-separated integers, the elements of the array A.
The next line contains N space-separated integers, the elements of the array B.

Output Format
In the single line of the output print the maximum possible sum of the elements x and y.

Constraints
1N106
1Ai,Bi103

Sample Input
5
2 9 4 3 8
1 5 6 2 11
Sample Output
20
Explanation

Note that we can choose x=9 from array A and y=11 from array B as they are co-prime. Also, this is the maximum possible sum that we can get.
